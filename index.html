ระบบการเรียนรู้ฟิสิกส์ ม.5 แบบละเอียดสมบูรณ์

บทเรียนเจาะลึก: การเคลื่อนที่แบบฮาร์มอนิกอย่างง่าย (SHM) และ คลื่นกล (Mechanical Waves)

การเชื่อมโยงเชิงมโนทัศน์: ทำไม SHM และ คลื่นกล ถึงสัมพันธ์กันอย่างแนบแน่น?

ในการศึกษาฟิสิกส์ ม.5 นักเรียนหลายคนมักแยกบทเรียน การเคลื่อนที่แบบฮาร์มอนิกอย่างง่าย (Simple Harmonic Motion - SHM) และ คลื่นกล (Mechanical Waves) ออกจากกันโดยสิ้นเชิง แต่ในความเป็นจริงแล้ว "คลื่นกลคือผลลัพธ์ของการส่งทอดพลังงานผ่านอนุภาคตัวกลางที่กำลังแกว่งตัวแบบ SHM อย่างต่อเนื่อง"

สะพานเชื่อมโยงที่ 1: การฉายเงาของการเคลื่อนที่แบบวงกลมสม่ำเสมอ

รากฐานของสมการ SHM ทั้งหมดมาจากการพิจารณาวัตถุที่หมุนเป็นวงกลมด้วยความเร็วเชิงมุมคงที่ ω เมื่อเราฉายแสงไฟจากด้านบนลงมา เงาของวัตถุบนแกน X จะเคลื่อนที่กลับไปกลับมาระหว่างจุดสองจุดที่เป็นแอมพลิจูดสูงสุด เงานั้นแหละคือประชากรตัวอย่างของการเคลื่อนที่แบบ SHM

สะพานเชื่อมโยงที่ 2: อนุภาคตัวกลางทุกลูกคือฟันเฟือง SHM

เมื่อคลื่นกลเคลื่อนที่ผ่านตัวกลาง เช่น คลื่นในเส้นเชือก พลังงานจะถูกส่งผ่านจากซ้ายไปขวา ทว่า เนื้อเชือกหรืออนุภาคแต่ละตัวไม่ได้วิ่งตามคลื่นไปตัวละครเหล่านั้นเพียงแค่ขยับขึ้นและลงอยู่กับที่ ณ ตำแหน่งเดิมของมัน การขยับขึ้นลงรอบจุดสมดุลเดิมนี้มีลักษณะตามเงื่อนไขของ SHM ทุกประการ! ดังนั้น:

  • ความถี่ของคลื่น (f) ก็คือ ความถี่ในการสั่นแบบ SHM ของอนุภาคแต่ละตัว
  • แอมพลิจูดของคลื่น (A) ก็คือ ระยะขยับจากจุดสมดุลสูงสุดของอนุภาคที่ทำ SHM
  • คาบการเคลื่อนที่ของคลื่น (T) ก็คือ เวลาที่อนุภาคตัวกลางใช้แกว่งครบรอบ SHM 1 รอบเต็ม
โครงสร้างปฏิสัมพันธ์: แหล่งกำเนิดทำ SHM → ถ่ายทอดพลังงาน → ตัวกลางสั่นแบบ SHM พลัดกันไปเชิงพื้นที่ → เกิดรูปคลื่นขยับ (v = fλ)

1. การเคลื่อนที่แบบฮาร์มอนิกอย่างง่าย (Simple Harmonic Motion)

นิยามอย่างเป็นทางการ: การเคลื่อนที่กลับไปกลับมาซ้ำทางเดิมผ่านตำแหน่งสมดุล โดยมีแรงดึงกลับ (Restoring Force) แปรผันตรงกับระยะขยับจากการกระจัด แต่มีทิศทางตรงกันข้ามเสมอ

1.1 สมการจลนศาสตร์ของ SHM (วิธีเต็มเชิงอนุพันธ์)

หากเริ่มต้นสังเกตการณ์ที่จุดสมดุลพอดี สมการการกระจัด ความเร็ว และความเร่ง ณ เวลา t ใดๆ จะเขียนได้ดังนี้:

การกระจัด: x(t) = A sin(ωt)
ความเร็ว: v(t) = ωA cos(ωt)
ความเร่ง: a(t) = -ω2A sin(ωt) = -ω2x

การวิเคราะห์เชิงลึก:

  • ที่ตำแหน่งสมดุล (x = 0): วัตถุจะไม่มีแรงดึงกลับ ทำให้ความเร่ง a = 0 แต่จะมีพลังงานจลน์สะสมสูงที่สุด ส่งผลให้ความเร็วมีค่ามากที่สุด vmax = ωA
  • ที่ตำแหน่งปลายสุด (x = ±A): วัตถุจะหยุดนิ่งชั่วขณะ ทำให้ความเร็ว v = 0 แต่แรงดึงกลับจากสปริงหรือเชือกจะตึงตัวที่สุด ทำให้เกิดความเร่งสูงสุดในทิศสวนทางกัน amax = ω2A

1.2 การประยุกต์ใช้กับระบบทางกายภาพ

1. ระบบมวลติดปลายสปริง (Mass-Spring System)
เมื่อมวล m ติดสปริงค่าคงที่ k แรงดึงกลับตามกฎของฮุคคือ F = -kx นำไปคำนวณหาความถี่เชิงมุมและคาบการแกว่งได้ว่า:

ω = √(k/m)  |  T = 2π√(m/k)

2. ระบบลูกตุ้มอย่างง่าย (Simple Pendulum)
เมื่อผูกมวล m กับเชือกยาว L แกว่งเป็นมุมแคบๆ (θ < 10°) แรงดึงกลับจะมาจากองค์ประกอบของแรงโน้มถ่วงสนามโลก g:

ω = √(g/L)  |  T = 2π√(L/g)

ข้อควรจำเชิงทฤษฎี: คาบของลูกตุ้มนาฬิกา ไม่ขึ้นกับมวลวัตถุเลย แกว่งมวล 1 kg หรือ 100 kg บนเชือกยาวเท่าเดิม คาบเวลาเท่าเดิมเสมอ!

2. คลื่นกล (Mechanical Waves)

นิยาม: การรบกวนตัวกลางเพื่อถ่ายโอนพลังงานจากจุดหนึ่งไปยังอีกจุดหนึ่ง โดยตัวกลางจำเป็นต้องมีสมบัติความยืดหยุ่น (Elasticity) เพื่อให้อนุภาคสั่นกลับมาที่เดิมได้

2.1 การจำแนกประเภทคลื่น

  • แบ่งตามการใช้ตัวกลาง: คลื่นกล (ต้องใช้ตัวกลาง เช่น คลื่นน้ำ คลื่นเสียง คลื่นในเส้นเชือก) VS คลื่นแม่เหล็กไฟฟ้า (ไม่ต้องใช้ตัวกลาง เช่น แสง รังสีเอกซ์)
  • แบ่งตามทิศทางสั่นของอนุภาค:
    - คลื่นตามขวาง (Transverse Wave): อนุภาคสั่นตั้งฉากทิศคลื่นวิ่ง (เช่น คลื่นเชือก, คลื่นน้ำ)
    - คลื่นตามยาว (Longitudinal Wave): อนุภาคสั่นขนานทิศคลื่นวิ่ง เกิดช่วงอัดและช่วงขยาย (เช่น คลื่นเสียง, คลื่นสปริง)

2.2 อัตราเร็วคลื่นและสมการหลัก

v = fλ = λ/T = s/t

ความเร็วคลื่นกล v จะขึ้นอยู่กับ คุณสมบัติของตัวกลางเท่านั้น (เช่น ความตึงเชือก, อุณหภูมิอากาศ, ความหนาแน่นของน้ำ) ถ้าตัวกลางเดิม ความเร็วคลื่นจะคงที่เสมอ ต่อให้เปลี่ยนความถี่ก็ตาม

2.3 เจาะลึกสมบัติคลื่น 4 ประการ (เน้นจุดออกสอบบ่อย)

1. การสะท้อน (Reflection)
กฎการสะท้อน: มุมตกกระทบ = มุมสะท้อน เสมอ วัดเทียบกับเส้นแนวฉาก (Normal Line)
จุดลัดกรณีคลื่นในเส้นเชือก:
- ปลายตรึงแน่น (Fixed End): คลื่นสะท้อนกลับมาเฟสเปลี่ยนไป 180 องศา (กลับหัวกลับหาง)
- ปลายอิสระ (Free End): คลื่นสะท้อนกลับมารูปร่างเหมือนเดิม เฟสไม่เปลี่ยน (0 องศา)

2. การหักเห (Refraction)
เกิดขึ้นเมื่อคลื่นเปลี่ยนผ่านตัวกลางต่างชนิดกัน ส่งผลให้ ความเร็ว (v) และความยาวคลื่น (λ) เปลี่ยนไป แต่ความถี่ (f) คงเดิมเท่ากับแหล่งกำเนิดเสมอ!
ใช้กฎของสเนลล์ (Snell's Law) ในการคำนวณ:

sin(θ1)/sin(θ2) = v1/v2 = λ1/λ2 = n2/n1

ทางลัดจำง่ายคดีคลื่นน้ำ: น้ำลึก (v มาก, λ ยาว, มุม θ กว้างขยายออกจากเส้นฉาก) → น้ำตื้น (v น้อย, λ สั้น, มุม θ แคบเบนเข้าหาเส้นฉาก)

3. การแทรกสอด (Interference)
เกิดจากคลื่น 2 ขบวนวิ่งมาเจอกันแล้วรวมกันตามหลักการซ้อนทับ (Superposition Principle) ออกสอบเฉพาะแหล่งกำเนิดอาพันธ์ (Coherent Sources) ที่มีเฟสตรงกัน
- แนวปฏิบัพ (Antinode: A): คลื่นเสริมกันสูงสุด (สันเจอสัน หรือ ท้องเจอท้อง)
- แนวบัพ (Node: N): คลื่นหักล้างกันสนิท (สันเจอท้อง)

แนวปฏิบัพ (เสริม): |S1P - S2P| = nλ  หรือ  d sinθ = nλ  (n = 0, 1, 2, ...)
แนวบัพ (หักล้าง): |S1P - S2P| = (n - 0.5)λ  หรือ  d sinθ = (n - 0.5)λ  (n = 1, 2, 3, ...)

4. การเลี้ยวเบน (Diffraction)
คือการที่คลื่นสามารถอ้อมผ่านสิ่งกีดขวางหรือเล็ดลอดผ่านช่องแคบ (Slit) ไปด้านหลังได้ตามหลักของฮอยเกนส์ (ทุกลำจุดบนหน้าคลื่นเสมือนเป็นแหล่งกำเนิดคลื่นใหม่)
- ถ้าช่องกว้าง d ≤ λ คลื่นจะเลี้ยวเบนได้ดีมาก หน้าคลื่นโค้งเป็นวงกลมโดยไม่มีแนวหักล้าง (ไม่มีแนวบัพเลย)
- ถ้าช่องกว้าง d > λ คลื่นจะเลี้ยวเบนและเกิดการแทรกสอดเกิดแนวบัพหักล้าง คำนวณหาแนวบัพด้วยสูตร: d sinθ = nλ

คลังสรุปสูตรฉบับเร่งด่วนและเทคนิควิธีลัดขั้นเซียน

สรุปสูตรเต็ม Simple Harmonic Motion

  • วิธีเต็ม จลนศาสตร์ทั่วไป:
    x = A sin(ωt)
    v = ωA cos(ωt) = ±ω√(A2 - x2)
    a = -ω2A sin(ωt) = -ω2x
  • วิธีเต็ม ความถี่เชิงมุมสปริง: ω = √(k/m)
  • วิธีเต็ม ความถี่เชิงมุมลูกตุ้ม: ω = √(g/L)
  • วิธีเต็ม พลังงานรวม SHM: Etotal = ½kA2 = ½mv2 + ½kx2

วิธีลัด เทคนิคพิชิตโจทย์ SHM

  • ลัดหา v และ a สูงสุด: จำทันทีไม่ต้องดิฟ vmax = ωA และ amax = ω2A
  • ลัดสปริงต่ออนุกรม: สปริงจะนิ่มลง ค่า k รวมลดลง คิดเหมือนตัวต้านทานขนาน → 1/kรวม = 1/k1 + 1/k2
  • ลัดสปริงต่อขนาน: สปริงจะแข็งขึ้น ช่วยกันดึง ค่า k รวมเพิ่มขึน → kรวม = k1 + k2

สรุปสูตรเต็ม คลื่นกล

  • วิธีเต็ม สมการพื้นฐาน: v = fλ = λ/T = s/t
  • วิธีเต็ม การหักเหสเนลล์: sinθ1/sinθ2 = v1/v2 = λ1/λ2
  • วิธีเต็ม แทรกสอดแนวปฏิบัพ (เฟสตรงกัน): |S1P - S2P| = nλ
  • วิธีเต็ม แทรกสอดแนวบัพ (เฟสตรงกัน): |S1P - S2P| = (n - 0.5)λ

วิธีลัด เทคนิคพิชิตโจทย์คลื่นกล

  • ลัดหาจำนวนแนวแทรกสอดทั้งหมดระหว่างแหล่งกำเนิด: นำระยะห่างระว่างแหล่งกำเนิด (d) หารด้วยความยาวคลื่น (λ) ได้ค่าเท่าไหร่ ให้ "ตัดเศษทศนิยมทิ้งทั้งหมด" เหลือแต่จำนวนเต็ม N
    - จำนวนแนวปฏิบัพ (Antinodes) ทั้งหมด = 2N + 1 แนว (รวมแนวกลาง A0)
    - จำนวนแนวบัพ (Nodes) ทั้งหมด = 2N แนว
  • ลัดเฟสต่างกัน: ทุกๆ ระยะทางที่เปลี่ยนไปเท่ากับ λ เฟสจะเปลี่ยนไป 2π เรเดียน (360 องศา) → สูตรด่วน Δφ = (2π / λ) × Δx
